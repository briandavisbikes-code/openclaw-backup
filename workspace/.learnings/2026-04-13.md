## 💡 Lessons Learned

1. **Weather location specificity**: wttr.in may resolve ambiguous location names incorrectly. Use explicit city/region for reliable forecasts.
2. **Discord token validation**: Numeric IDs are not valid bot tokens. Ensure token is a proper string.
3. **Guild allowlist configuration**: Missing guild allowlist with `requireMention: false` can cause bot to ignore messages. Must configure `groupPolicy: "allowlist"` with guild entry.
4. **Ollama web‑search endpoint**: Not all Ollama installations have web‑search enabled. Verify capabilities before relying on experimental endpoints.
5. **Provider migration**: Always test API connectivity and cost tracking after switching primary LLM providers.
